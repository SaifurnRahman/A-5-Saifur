Question-1: What are the different ways to select an element in the DOM?

There are many way that you can select an element in the DOM. like getElementById, getElementByClassName, getElementByTagName, querySelector, querySelectorAll etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the content of an element including tags. innerText returns only the visible text inside the tags. textContent returns all text inside and element including hidden text

Question-3: What is event delegation in the DOM?

Event Delegation is a trick where you add one event listener to a parent element instead of adding it to multiple child elements. The parent listens for events from its children.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event starts from the element that was clicked and moves up to its parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript

for create we use "document.createElement("tagName")", for add we use .appendChild(), for remove element we use .remove() or .removeChild()
